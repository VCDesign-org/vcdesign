# Positioning of VCDesign Core

## Why Minimal?
Because the Problem (UVW) is universal and irreducible, the Response (Core) must be equally universal and irreducible.

## The "Stop or Review" Principle
In a world where value decays (UVW), the only honest response when continuity is lost is to **Stop**.
Continuing without check is to process noise as signal.

- **We do not prescribe** *how* to judge value.
- **We only prescribe** *what to do* when judgment confidence is lost.
