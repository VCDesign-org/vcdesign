# Axioms of UVW

These are the starting observations. They are treated as given conditions, not design choices.

## A1. Value Instability
Value is not an intrinsic property of an artifact; it is a relationship between an artifact and a satisfying context.

## A2. Context Drift
The context that makes a value valid changes over time (technological shift, social shift, user belief shift).

## A3. Judgment Decay
A judgment made at time $T_0$ loses confidence as time proceeds to $T_1$, unless re-verified.

## A4. Non-Self-Evidence
The intent or value of an artifact is not self-evident from the artifact alone; it requires an external interpretive frame (Binding).

## A5. Responsibility Gap
Machines process logic, but only humans (or legal entities) can bear responsibility for the outcome of that logic.
